Gerenciamento de Imagens
docker pull <imagem>
Baixa uma imagem do Docker Hub para o cache local
docker images
Lista todas as imagens no cache local
docker rmi <imagem>
Remove uma imagem (se não estiver em uso)
docker build -t nome:tag .
Constrói uma imagem a partir de um Dockerfile
Gerenciamento de Containers
docker run <imagem>
Cria e inicia um container
docker run -d <imagem>
Executa em background (detached)
docker run -p 8080:80 nginx
Mapeia porta do host para o container
docker run --name meu nginx
Atribui um nome personalizado ao container
docker run -v vol:/path img
Monta um volume no container
docker run -e VAR=val img
Define variável de ambiente
docker run --network net img
Conecta container a uma rede específica
docker ps
Lista containers em execução
docker ps -a
Lista todos os containers (incluindo parados)
docker stop <id|nome>
Para um container em execução
docker start <id|nome>
Inicia um container parado
docker rm <id|nome>
Remove um container parado
docker logs <id|nome>
Exibe logs do container
docker exec <nome> <cmd>
Executa comando dentro de um container
Redes
docker network ls
Lista todas as redes
docker network create <nome>
Cria uma nova rede bridge
docker network rm <nome>
Remove uma rede (se não estiver em uso)
Volumes
docker volume ls
Lista todos os volumes
docker volume create <nome>
Cria um novo volume gerenciado
docker volume rm <nome>
Remove um volume
Docker Compose
docker compose up
Sobe todos os serviços do compose (usa o editor)
docker compose down
Derruba todos os serviços do compose
